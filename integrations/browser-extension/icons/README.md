# Extension Icons

Place your icon files here:
- icon16.png (16x16) - Toolbar icon
- icon48.png (48x48) - Extension management
- icon128.png (128x128) - Chrome Web Store

## Quick Icon Generation

You can create icons using:
1. Online tools: https://www.favicon-generator.org/
2. ImageMagick: `convert logo.png -resize 16x16 icon16.png`
3. Figma/Sketch export

## Temporary Placeholder

For now, you can use a simple colored square or emoji as a placeholder.
The extension will still work without custom icons.
