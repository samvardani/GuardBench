- ✅ Settings page with proper WordPress standards
- ✅ Test page for debugging

## Installation
1. Upload `wp-ai-engine-pro-simple-v4.0.2.zip` to WordPress
2. Activate the plugin
3. Go to **AI Engine** in WordPress admin menu
4. Configure your OpenAI API key in Settings
5. Add `[wpai_chatbot]` shortcode to any page
